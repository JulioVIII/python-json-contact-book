# Python JSON Contact Book

A simple contact book written in Python using JSON.

## Features

- Add contacts
- View contacts
- Save contacts to a JSON file
- Load contacts from a JSON file
- Input validation
- Console application

## Technologies

- Python 3
- JSON

## How to Run

```bash
python json_contact_book.py
```

## Example

```text
--- JSON CONTACT BOOK ---
1. Add contact
2. View contacts
3. Save contacts
4. Load contacts
5. Exit
```
